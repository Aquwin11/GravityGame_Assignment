## Overview

This Unity project implements an advanced third-person gravity-manipulation controller, allowing the player to walk on walls, ceilings, and dynamically rotate gravity in real time.



* Custom gravity direction system
* Player movement aligned to surfaces
* Point collection with event-based UI updates
* Win/Lose game logic with timer



## Project Structure



#### Player Movement \& Rotation (PlayerController.cs)

* Basic Third-person controller
* Change gravity based on the player Transform using arrow keys
* Walk on Surface and ceiling and wall and snap to surfaces



#### Camera System (CameraController.cs)

* Smooth damp movement
* Look rotation
* Camera collision





#### Game Manager (GameManager.cs)

* Timer countdown
* Win \& Lose Condition
* End-game UI



#### Miscellaneous (InpuManager.cs \& BorderScript.s)

* Handle Condition when goes through the wall
* Using Unity Enhanced input system created a input mapping
